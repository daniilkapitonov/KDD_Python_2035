**Домашнее задание после урока №2**


1.Напишите программу, которая считывает три строки. Если эти три строки: “красный”, “жёлтый”, “зелёный” или “3”, “2”, “1”, то программа выводит “ПОЕХАЛИ”, если нет, то “СТОЙ”


2.Написать программу, чтобы она определила в каком интервале находится число и вывела соответствущее сообщение. 
[-20,-10] - очень холодно
(-10,0) - холодно
[0,10] - уже можно пить воду на улице
(10,20) - скоро купаться
[20+] - идём купаться

Число вводится с клавиатуры.
